 PLSearch\+CPP is a C++ library that approximates Prolog predicates using a continuation-\/passing style. It provides a framework for defining and executing Prolog-\/like predicates, including support for backtracking, choice points, and unification.\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item Prolog-\/like predicates
\item Continuation-\/passing style execution
\item Backtracking and choice points
\item Unification of terms
\item Support for variables, integers, floats, atoms, and lists
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_README_autotoc_md2}
